Import new posts, pages, custom post type AND edit existing posts, pages and custom post type via WP Importer from a CSV file.

== Description ==

WP Importer plugin provide an option for create new pages using CSV (Comma Separated Value) import functions. CSV (Comma Separated Values) is a file format for data storage which looks like a text file. The information is organized with one record on each line and each field is separated by comma. Using csv file user can easily manage page content without create/update pages, there will no need to do changes manualy. Your all pages content will be update in a few seconds after click on a "Import Now" button, it will save your valuable time :)

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The plugin will be available under Tools -> WP Importer on
    WordPress administration page.
